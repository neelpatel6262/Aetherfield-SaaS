## Why I Created This Project

In early 2025, I wanted to seriously improve my **React + modern frontend development skills** — especially:

- Converting high-fidelity Figma designs into pixel-perfect, responsive, animated code  
- Writing clean, reusable, maintainable React components  
- Mastering scroll-triggered animations (without making them distracting)  
- Building real-world routing patterns (nested layouts, dynamic params, 404)  
- Creating smooth UX features like password strength meters, full-screen mobile menus, form validation  

Instead of following a step-by-step tutorial (which I already did many times), I decided to **challenge myself** by picking a **random professional SaaS template** from the Figma Community and rebuilding it **100% from scratch** — no starter code, no copying.

## What I Learned / Improved While Building

- **React Router v6 deep dive** — nested `<Layout />`, dynamic params (`:slug`, `:jobId`), 404 handling, `ScrollToTop` component
- **Reusable animation system** — created `useScrollAnimation` custom hook → DRY scroll-triggered fade + slide animations everywhere
- **Full-screen mobile menu** with backdrop blur, click-outside close, smooth slide animation
- **Form UX** — password strength meter, show/hide toggle (fixed event propagation bug), real-time validation, terms checkbox
- **LocalStorage simulation** — save user data after signup + console.log object
- **Consistent typography & design tokens** — custom fonts (heading, subheading, btn), responsive clamp() font sizes
- **Stagger animations** — cards/links appear one by one (0.08s–0.15s delay)
- **Performance & clean code** — no repeated props, reusable components (sections, layout), proper folder structure
- **Responsive images** — constrained logo size on mobile, object-cover/contain usage
- **Professional README writing** — storytelling, structure, growth reflection
